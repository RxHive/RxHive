# Taurine + Acetylcysteine

* **Acetylcysteine:** Used medically as a mucolytic agent (to thin mucus) in respiratory conditions like bronchitis and cystic fibrosis. It also has antioxidant properties and shows some promise in treating paracetamol (acetaminophen) overdose.
